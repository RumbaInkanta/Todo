# TODO приложение

## О проекте  
Это лёгкое приложение «списка задач» (To-Do), реализованное на Python и Kivy/KivyMD, с использованием базы данных для хранения задач.  
**Основные технологии:** Python, Kivy/KivyMD, SQLite (или другая БД)  
**Цель:** предоставить стабильный и расширяемый старт для управления задачами с мобильным или десктоп-интерфейсом.

## Возможности  
- создание, редактирование и удаление задач;  
- группировка задач/категорий (если предусмотрено);  
- сохранение задач в локальной базе данных;  
- пользовательский интерфейс, реализованный через Kivy/KivyMD (.kv-файлы + Python-логика);  
- модуль тестирования (есть файлы `test_list.py`, `test_task.py`, `test_storage.py`).

Использование
При первом запуске приложение создаст (или подключится к) базе данных.

В интерфейсе можно добавлять задачи, помечать их как выполненные, удалять или изменять.

Все данные сохраняются локально — можно расширить функционал: синхронизацию, экспорт/импорт, категории, напоминания и т.д.
